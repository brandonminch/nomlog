# User Interface & Experience

[← Feature index](../FEATURES.md)

## Core Features

- **Mobile-First Design**
  - React Native with Expo
  - iOS and Android support
  - Safe area handling
  - Responsive layouts

- **Navigation**
  - Bottom navigation bar (Home, Stats, Settings)
  - Floating action button for quick meal logging
  - Modal-based interactions
  - Smooth transitions and animations

- **Visual Components**
  - Meal log cards with nutrition summaries
  - Macro progress wheels
  - Weekly statistics bar charts
  - Water tracker visualization
  - Loading states and error messages
  - Typing indicators for AI responses

- **Date Management**
  - Timezone-aware date handling
  - 7-day carousel for meal history
  - Automatic day change detection
  - Date picker for editing meal times
