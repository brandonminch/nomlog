## Usage in React Native

All of these icons can be imported and used in your React Native app like this:

```typescript
import { 
  Apple, 
  Coffee, 
  Utensils, 
  Pizza,
  // ... etc
} from 'lucide-react-native';

// Then use in your component
<Apple size={24} color="#000" />
```

## Notes

- This list is based on Lucide icon library version 0.544.0 (the version currently installed in your project)
- According to [Lucide's official categories page](https://lucide.dev/icons/categories#food-beverage), there are **69 food & beverage icons** in total
- Some icons may have been added in newer versions
- The "-off" variants are useful for indicating dietary restrictions or allergens
- Lab/Experimental icons may require the `@lucide/lab` package and may not be available in `lucide-react-native`
- All icons are customizable with props like `size`, `color`, `strokeWidth`, etc.
- Icons currently used in your app: `Coffee`, `Utensils`, `UtensilsCrossed`, `Dumbbell`, `Wheat`, `Droplet`, `GlassWater`
